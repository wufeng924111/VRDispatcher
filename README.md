# Demo


## Example

一种新的swift组件化中间件调度方式（基于枚举），具体可参照Example，中间件代码在VRDispatcher库里面，VRAModule和VRBModule是示例模块.

## Requirements

## Installation

Demo is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Demo'
```
